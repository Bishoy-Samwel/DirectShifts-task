**Environment**  

- Ruby version: 3.0.1
- Rails version: 7.0.1
- Default Port: 3000

**Setup Database**
- run: 
```bash
rails db:create
rails db:migrate
```

**Commands**
- install:
```bash
npm i
```
- run: 
```bash
./bin/dev
```
- install: 
```bash
bundle install
```
